Pisi Linux Team proudly presents Pisi Linux 1.0

Pisi Linux is an operating system, based on GNU/Linux and Pisi.
It is created as a successor of Pisi-based Pardus Linux, after its End Of Life declaration. Pisi Linux contains several unique technologies:
COMAR (COnfigurationMAnageR), designed to facilitate the integration and co-existence of software.
MUDUR, designed to streamline the boot process.
PISI (Packages Installed Succesfully, as Intended), is Turkish for kitty and was the name given to the custom packaging system developed for Pardus Linux.
YALI (Yet Another Linux Installer), the custom installer.
KAPTAN, is Turkish for captain and is the desktop greeter, which start at the first start. It allows a user to change the settings.
PANDA, video-driver manager.

The model of distribution of Pisi Linux is rolling release.

System Requirements
64-bit processor
At least 2 GB memory and 12 GB disk space for system
It is recomended to create swap space for Pisi Linux installation.

Multimedia
Codecs for playing audio and video files are included.
There is only 64bit version of Pisi Linux.
Packages for all necessary 32bit applications (Skype, Steam) are included.
